Build Instructions (Athena with Makefile)
==================
(cd into starter4, aka main file)
$ mkdir build
$ cd build
$ cmake ..
$ make
then run any of the given test cases. 

For example:

./a4 -input ../data/scene05 bunny 200.txt -output05.png -size 200 200
